平台介绍

威胁情报平台(TIP)是一个集成化的解决方案，用于收集、分析、存储和共享网络安全威胁情报。该平台帮助安全团队高效管理威胁数据，提供快速响应能力。

主要功能包括：

  • 多源情报数据收集与整合
  • 标准化的威胁情报存储
  • 高级搜索与分析功能
  • 灵活的情报共享机制
  • 集成化的案例管理

API文档

平台提供了完整的RESTful API，支持所有主要功能的编程访问。

API端点示例

  • GET /api/iocs/ip/{address} - 查询IP情报
  • GET /api/iocs/domain/{name} - 查询域名情报
  • POST /api/cases/ - 创建新案例
  • GET /api/search?q={query} - 全文搜索

所有API请求需要包含适当的认证信息，支持JWT和API Key认证方式。
